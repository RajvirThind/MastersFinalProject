# Installation

## Stable release

To install BESSOptimisation, run this command in your terminal:

```sh
uv add BESSOptimisation
```

Or if you prefer to use `pip`:

```sh
pip install BESSOptimisation
```

## From source

The source files for BESSOptimisation can be downloaded from the [Github repo](https://github.com/RajvirThind/BESSOptimisation).

You can either clone the public repository:

```sh
git clone git://github.com/RajvirThind/BESSOptimisation
```

Or download the [tarball](https://github.com/RajvirThind/BESSOptimisation/tarball/master):

```sh
curl -OJL https://github.com/RajvirThind/BESSOptimisation/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd BESSOptimisation
uv pip install .
```
